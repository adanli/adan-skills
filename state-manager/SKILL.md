# 状态机生成器

## 什么时候使用

当用户提出“帮我生成状态机”、“帮我构建状态机”时，必须使用这个技能

## 工作流程

1. 解析用户提供的信息
   - 如果用户提供的是领域模型图，将其中的聚合根、实体，作为状态机的对象
   - 如果用户提供的是数据模型图，将其中的概念、实体，作为状态机的对象
2. 判断在用户提供的原始信息中，对象中是否设计了状态，或者类似状态的属性，如果未设计，提示用户“XX对象不存在状态属性，跳过为该对象绘制状态机”，并继续为下一个对象绘制状态机
3. 如果用户提供了用例图，或者功能清单，结合其中对功能的描述信息，帮我绘制对应对象的状态机，包括：对象、对象的状态、以及状态之间流转的行为

### 核心技巧

1. 在绘制状态机图的时候，使用支持mermaid的语法绘制
